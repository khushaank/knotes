Security Policy

We take the security of K. Notes seriously. Our backend is powered by Supabase, which provides robust security features.

Vulnerability Reporting

If you find a security vulnerability, please contact us immediately at security@knotes.com. We will investigate and respond as quickly as possible.

Authentication

User authentication is handled via Supabase Auth, which uses secure session tokens. We recommend using a strong, unique password for your account.
